# Finance Tracker: Backend

## Intro
Finance Tracker is a web application that allows users to track their expenses and income. Set their saving goals and budgeting plans.
The backend is built using Node.js, Hono, Prisma, Better Auth, and SQLite.

## Tech Stack

- Node.js
- Hono
- Prisma
- Better Auth
- SQLite

```sh
pnpm install
pnpm run dev
```

```sh
open http://localhost:8000
```
