# Finance Tracker

## Intro

Finance Tracker is a web application that allows users to track their expenses and income. Set their saving goals and budgeting plans.
The project follows the client-server architecture; the front-end and back-end are separated into different repositories. PNPM workspaces
manages scripts and dependencies.

## Tech Stack

- Front-end: TBD
- Back-end: Node.js with Hono.js

## Scripts

```sh
pnpm run dev
```
 Runs both the front-end and back-end servers in development mode.
 Checkout the package.json file for more commands.
